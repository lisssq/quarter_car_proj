## Описание проекта

Проект реализует **двухмассовую модель "четверти автомобиля"** для анализа влияния параметров подвески на поведение автомобиля при преодолении дорожных неровностей. Основная цель - найти оптимальные значения жесткости и демпфирования подвески, минимизирующие общий критерий качества.

### Критерии качества:
- **Комфорт** (минимизация ускорения кузова);
- **Управляемость** (минимизация деформации шины);
- **Ход подвески** (ограничение максимального хода).

## Математическая модель

Модель основана на системе дифференциальных уравнений движения двух масс:

`m1 * z1'' = k2*(z2-z1) + c2*(z2'-z1') - k1*(z1-z0)`

`m2 * z2'' = -k2*(z2-z1) - c2*(z2'-z1')`

где:
- `m1` - неподрессоренная масса (колесо)
- `m2` - подрессоренная масса (кузов)
- `k1` - жесткость шины
- `k2` - жесткость подвески (оптимизируемый параметр)
- `c2` - коэффициент демпфирования (оптимизируемый параметр)
- `z0` - дорожное возмущение

## Функциональность

### 1. Анализ влияния параметров
- Сравнение 5 типов подвесок: мягкая, средняя, жесткая, со слабым и сильным демпфированием;
- Визуализация временных характеристик;
- Расчет критериев качества для каждой конфигурации.

### 2. Оптимизация параметров
Реализовано 3 метода оптимизации:
- **Nelder-Mead**;
- **Hyperopt**;
- **Optuna**.

### 3. Визуализация результатов
- Графики временных характеристик;
- Сравнение оптимальных решений;
- Анализ эффективности методов оптимизации.
